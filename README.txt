UNITMETRA SITE V3

Что работает:
- адаптивная многостраничная версия;
- RU/EN для основных интерфейсных элементов;
- мобильное меню;
- модальное окно диагностики: Telegram, MAX, форма;
- валидация файлов до 10 МБ;
- локальное сохранение заявок и демонстрационная панель /admin/;
- FAQ, вкладки примеров отчётов, анимации;
- ссылки на Excel/PDF/HTML демонстрационного кейса;
- SEO-страницы, robots.txt, sitemap.xml, JSON-LD;
- cookie-баннер и заготовка целей Яндекс Метрики.

Перед публикацией:
1. Купить и подключить домен.
2. Подключить хостинг (например, Netlify/Vercel) и рабочую CMS.
3. Настроить серверное хранение заявок/файлов и уведомления на почту.
4. Добавить ID Яндекс Метрики.
5. Вставить реквизиты самозанятой в политику.
6. Загрузить реальный скриншот отзыва.
7. Проверить домен и товарный знак Unitmetra.

Локальный просмотр: запустить в папке команду python -m http.server 8000 и открыть http://localhost:8000
